Question 1: Difference Between Map, filter, forEach, find ?

ForEach runs code on every element of an array by calling a function and return undefined.

Map runs code on every element of an array by calling a function and return result in a new array.

Filter runs code on every element of an array by calling a function to see if it meets any condition and return results in a new array that meets the given condition. It will return an empty array if the condition is not fulfilled or false.

Find runs code on every element of an array by calling a function to see if it meets any condition and return the first element of the array that meets the given condition. It will return an undefined if the condition is not fulfilled or false.

Question 2: Difference Between localStorage, sessionStorage and cookies ?

Local storage is a way to store data with no expiry date. It store data untill the data is deleted manually. It doesn't send data to the server side.It is only meant for client side.

Session storage is a way to store data only for the running session. When browser or tab is closed, data will get deleted. It is also meant for only client side.

Cookies are small piece of data.Cookies cannot be more than 4KB.Cookies store small piece of data and send the data back to the server.Cookies usaully store user preference data and meant for server side reading.

Question 3: What is Event Loop?

Event Loop create a stack of event and execute the code line by line. It is also called asyncronous way of executing code. When any function is executed, the function is popped from the stack.When any new function is added it is sent to the queue of the stack to execute. Event Loop maintains the correct sequence of the stack for execution.
